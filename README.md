# Проведённая работа
В контексте данного проекта был реализован геймплей для прототипов 2D и 3D игр и проведено знакомство с основными составляющими рабочего процесса:
* Работа с самим редактором Unity и его функционалом, его базовая настройка
* Работа с аудиосистемой
* Работа со скриптами в Unity Editor
* Написание скриптов **(путь: Assets/_Unity Essentials/Scripts)**
* Работа с 3D и 2D объектами и компонентами
* Публикация web билда проекта на play.unity.com
* Работа в самом редакторе Unity с составляющими проект классами и объектами

Проект можно собрать в Unity
